# Sentiment-analysis
This project uses the IMDb dataset to classify movie reviews as Positive or Negative using BERT.

🛠️ Technologies Used
Hugging Face Transformers
Datasets Library
PyTorch
Streamlit

How to Run
1. Install dependencies

pip install -r requirements.txt

2. Train the model

python train.py

3. Run the app
 streamlit run app.py

📊 Dataset

IMDb dataset loaded from Hugging Face.

🎯 Output
Input: Movie review text
Output: Positive / Negative sentiment
